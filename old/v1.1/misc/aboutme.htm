DETAILS OF UNIVERSITY PROJECTS

 

Fachhochschule München(FHM)

Final Year Project: Creation of a 3D model of an unmanned flying vehicle using PRO/Engineer.

 

3D model creation of an airplane that had already been designed and built by the FHM. First, the dimensions of the plane were taken using a measuring table.  Then, Pro/Engineer, running on a Silicon Graphics workstation under X-Windows, was used to build the 3D model.  The purpose of this project was to create a 3D computer model that can be used by other people and from which the blue prints can be generated. Other students are now able to design new parts without needing access to the actual airplane.

 

University of Southampton

Group Design Project: Design of an autonomous attitude control system for small satellites based on a CCD star tracker.

 

Working as part of a group of five, my task was the design of a lunar survey mission, the definition of the overall system performances and the design of the general layout of the spacecraft.  After defining the mission concept and the orbit strategy, the budgets for mass, power and communication were computed.  This was done using a program developed in C. Finally the spacecraft pointing accuracy necessary for this mission was defined and the optimal location of the orbit and attitude control thruster was computed.
