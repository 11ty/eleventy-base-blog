Sokoban  

This is an html version of Sokoban. Have a play around, it's fairly simple, since I am still working on it. Excellent graphics - I think not!
Move Soko by clicking on the grid or by clicking on the navigation arrows to the left of the grid. It takes a second or two to load each level.
Please email any levels you may want to donate to me. All donors will be named on this page.

This only works in Internet Explorer. Nestcape Users: I don't really have the time or will to make this Netscape compatible. The only way I can think of doing his (and admittedly I haven't given it much thought) is by using loads of images and "A" tags. If you can think of better way, do get in touch. Else use Internet Explorer which is a much better browser nowadays - and I used to be a strict Netscape user not so long ago.
